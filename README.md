# Dango Shop

Dango Shop is a store selling accessories and gifts.

## Backend

Step 1:

```bash
cd backend/
npm install
```

Step 2:

```bash
linux: touch .env
or
windows: type nul > .env
```

Step 3: Fill in the fields to the file .env

```bash
API_URL = /api/v1
CONNECTION_STRING = ...
SECRET_KEY = ...
```

Step 4:

```bash
npm run start
```

## Frontend

Step 1:

```bash
cd frontend/
npm install
npm install @angular/cli -g
npm install nx -g
```

Step 2:

```bash
npm run start
or
nx serve
```
